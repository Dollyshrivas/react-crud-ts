# React CRUD Users (TypeScript + Tailwind CSS)

A modern **CRUD (Create, Read, Update, Delete)** application built with **React**, **TypeScript**, **Tailwind CSS**, and **Axios**.  
The app fetches users from the [JSONPlaceholder API](https://jsonplaceholder.typicode.com/users) and allows you to create, update, and delete users (simulated).
